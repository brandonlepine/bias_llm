Reviewing BBQ stereotype pairs

Each card is one unique stereotype pair — it already covers all names and both frames (the name frame “Alex is X and …” and the group frame “X people are …”). Judge it once; your decision propagates to every variant.

Pick one action, optionally type a corrected predicate / note, then press the green ✓ (accept) to save.

  • ✓ Approve — a valid, clean stereotype contrast (natural, real stereotype, right groups).
  • ✗ Reject — drop it (not a real stereotype, ungrammatical, or wrong groups).
  • 🔍 Mark for review — unsure; revisit later.
  • ✏ Mark for edit — keep the stereotype but fix the predicate; type the corrected predicate in the box (e.g. a habitual criminal). It rewrites both frames.

Target group is shown in red, the dominant/reference group in blue.
